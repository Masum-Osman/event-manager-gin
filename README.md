## Event Management Service For 10 MS

> Project run instructions 

Install all the dependencies using below command 
```shell
go get
```
Then build project
```shell
make build
```
Finally, run the binary file
```shell
make run_bin
```

---

> Project build with docker

```shell
make docker_build
```

```shell
make docker_run
```
